# API Specifications & Overrides

This directory contains the API specification files used as the source of truth
for generating type-safe Swift structures.

## Document List

*   [firebasevertexai_public_openapi3_0_v1beta.json](firebasevertexai_public_openapi3_0_v1beta.json):
    The base OpenAPI 3.0.3 specification JSON file describing the Firebase AI
    Logic API.
*   [firebasevertexai-overrides.yaml](firebasevertexai-overrides.yaml):
    The manually maintained YAML overrides file defining `required` properties
    and flat `oneOf` unions to improve Swift type safety.
*   [firebasevertexai-openapi.yaml](firebasevertexai-openapi.yaml):
    The upgraded and patched OpenAPI 3.1.0 specification YAML file generated by
    `upgrade_spec.py`. This file is the direct input to `generate_types.py`.

---

## Build Pipeline Integration

The specification upgrade script merges
`firebasevertexai_public_openapi3_0_v1beta.json` and
`firebasevertexai-overrides.yaml` to produce `firebasevertexai-openapi.yaml`.

Do not modify `firebasevertexai-openapi.yaml` directly; instead, make changes
to `firebasevertexai-overrides.yaml` and run:

```bash
python utilities/scripts/upgrade_spec.py
```
